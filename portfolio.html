GIFTASTIC

Descritpion:

I used the GIPHY API to make a dynamic web page that populates with gifs of famous female pop stars.When the user clicks on a button, the page grabs 10 static, non-animated gif images from the GIPHY API and places them on the page. When the user clicks one of the still GIPHY images, the gif animates. If the user clicks the gif again, it stops playing.

Technologies:

Giphy API, JSON, Javascript & JQuery

Train Schedule

Descritpion:

Created a train schedule application that incorporates Firebase to host arrival and departure data. The app retrieves and manipulates the information with Moment.js. It provides up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station.

Technologies:

Javascript, Firebase & JQuery

LIRI

Description:

LIRI is a command line node app that takes in parameters and gives you back data. The parameters are 'My Tweets', 'spotify this song' and 'movie this' and the application gives you back …

  • a) Your latest 20 tweets
